***-->it gives you a nodelist of li ..but it's not array.you can use foreach loop in nodelist
  
  templist[0].style.color="white"

***--> it turn color white of the first li element(home).

***-->using foreach loop in nodelist:  

  templist.foreach((list-element)-> {
    list-element.style.color="green"
  })

const tempLiList = Document.getElementByClassName('list-item')

***--> it gives you a HTMLcollection and you cannot use foreach loop in HTMLcollection.you have to convert HTMLcollection to an array ...after that you can use foreach loop

***-->how to convert HTMLcollection or nodelist in to an array 

const myConvertedArray = Array.from(tempLiList)

***-->now, tempLiList HTMLcollection is converted in an array

***-->using foreach loop in my myConvertedArray... ... ...

myConvertedArray.foreach(function(list-value){
    list-value.style.color="orange";
    list-value.style.padding = "15px";
    list-value.style.border = "1px solid green ";
} )
